# Knowledge Chatbot (RAG)

A Retrieval-Augmented Generation (RAG) chatbot that answers questions using external knowledge stored in a vector database.

## Components

- build_vector_db.py
- ragchat.py
- rag_api.py

## Workflow

Documents
↓
Embeddings
↓
Vector Database
↓
Retriever
↓
LLM Response

## Technologies

- Python
- RAG
- Embeddings
- Vector Search
